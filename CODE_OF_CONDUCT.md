# Code of Conduct

## Our Pledge

We are committed to providing a welcoming, respectful and inclusive environment for everyone.

## Expected Behaviour

- Be respectful.
- Be constructive.
- Be patient.
- Assume good intentions.
- Help others learn.

## Unacceptable Behaviour

- Harassment or discrimination.
- Personal attacks.
- Trolling.
- Spam.
- Deliberate disruption.

## Enforcement

Project maintainers may edit, remove or reject contributions that do not follow this Code of Conduct.
